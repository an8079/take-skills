## 调试模式示例

### 启动调试

```
/debug
```

### 输出示例

```
🐛 调试模式

当前问题: 用户登录后Token验证失败

## 调试步骤

1. 检查Token生成逻辑
2. 检查Token验证逻辑
3. 检查Token存储

让我开始排查...

### 排查结果

发现问题了！

在 `src/auth/token.ts:45`:
```typescript
// 错误：使用同步方法
const token = jwt.sign(payload, secret);

// 正确：应该使用异步方法
const token = await jwt.sign(payload, secret);
```

是否需要我帮你修复？
```

## 工作流程示例

### 完整项目流程

```
1. 启动访谈
   /interview

2. 完成访谈后写规格
   /spec

3. 创建计划
   /plan

4. 按计划编码
   /code

5. 测试验证
   /test

6. 代码审查
   /review

7. 调试修复（如需要）
   /debug
```

### 快速迭代流程

```
1. 快速开始
   快速开始

2. 直接计划（跳过详细规格）
   /plan

3. 开始编码
   /code
```
