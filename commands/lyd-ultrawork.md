# /ultrawork - Ultrawork 并行执行模式

最大化并行 Agent 执行，适合批量修复和重构任务。

## 使用方式

```
/ultrawork "任务描述"
```

或

```
ULTRAWORK
极限工作
并行执行
```

## 工作模式

```
┌──────────────────────────────────────────────────────┐
│                 Ultrawork Engine                      │
├──────────────────────────────────────────────────────┤
│                                                      │
│   Input Task                                         │
│       │                                               │
│       ▼                                              │
│  ┌─────────────┐                                     │
│  │ Task Split │  将任务分解为独立子任务             │
│  └──────┬──────┘                                     │
│         │                                             │
│         ▼                                             │
│  ┌─────────────────────────────────────────────┐    │
│  │           Parallel Execution                 │    │
│  │  ┌─────────┐ ┌─────────┐ ┌─────────┐       │    │
│  │  │ Agent 1 │ │ Agent 2 │ │ Agent N │ ...  │    │
│  │  │ Task A  │ │ Task B  │ │ Task C  │       │    │
│  │  └─────────┘ └─────────┘ └─────────┘       │    │
│  │       │           │           │            │    │
│  │       └───────────┴───────────┘            │    │
│  │                   │                         │    │
│  │              Results Merge                   │    │
│  └───────────────────┬─────────────────────────┘    │
│                      │                               │
│                      ▼                               │
│              ┌───────────────┐                       │
│              │ Result Review │                       │
│              │   汇总报告    │                       │
│              └───────────────┘                       │
│                                                      │
└──────────────────────────────────────────────────────┘
```

## 适用场景

| 场景 | 说明 |
|------|------|
| **批量 Bug 修复** | 多个独立 bug 同时修复 |
| **代码重构** | 多个文件同时重构 |
| **测试生成** | 多个模块同时生成测试 |
| **文档更新** | 多个文档同时更新 |
| **依赖升级** | 多个包同时升级 |

## 特点

| 特点 | 说明 |
|------|------|
| **最大并行** | 尽可能多的任务并行执行 |
| **任务独立** | 子任务之间无依赖 |
| **结果汇总** | 自动合并所有结果 |
| **错误隔离** | 单个任务失败不影响其他 |

## 使用建议

```
# 适合 ultrawork 的任务
/ultrawork "修复项目中所有 ESLint 警告"
/ultrawork "为所有 API 路由添加单元测试"
/ultrawork "升级所有依赖包到最新版本"

# 不适合 ultrawork 的任务（有依赖）
/ultrawork "重构整个登录流程"  # 任务之间有依赖，不适合
```

## 与 /ralph 的区别

| 维度 | /ultrawork | /ralph |
|------|------------|--------|
| 并行度 | 最大并行 | 串行为主 |
| 循环 | 单次执行 | 持久循环 |
| 适用 | 批量独立任务 | 复杂迭代任务 |

---

**提示：** 使用 `/ultrawork` 当你有多个独立任务需要同时处理时，它会最大化并行执行效率。
