## 更新说明

### 如何使用本文件

1. AI 自动记录：当 AI 识别到用户纠正时，自动添加到「待处理的纠正」表格
2. 用户触发反思：用户说 `/reflect` 时，AI 处理所有待反思的纠正
3. 规则应用：
   - 全局规则 → 更新 `CLAUDE.md`
   - 项目规则 → 更新此文件的「项目规则」
   - 领域规则 → 更新对应 `SKILL.md`

### 纠正检测模式

当用户说以下内容时，AI 自动识别为「纠正」：

| 模式 | 示例 |
|------|------|
| 否定 + 正确做法 | 「不对，用 X 不是 Y」「别用 A，用 B」 |
| 实际上... | 「实际上应该...」「其实是...」 |
| 记住... | 「记住：...」「以后要...」 |
| 这样不行 | 「这样不行，应该...」「错了，要...」 |
