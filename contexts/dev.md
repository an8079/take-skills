# 开发模式上下文

在开发模式下，专注于实现功能，遵循以下原则。

## 当前模式：开发

## 优先级

1. **功能正确** - 确保代码实现预期功能
2. **代码质量** - 遵循代码规范和最佳实践
3. **测试覆盖** - 为新代码添加测试
4. **文档更新** - 更新相关文档

## 开发流程

1. 阅读任务描述和规格文档
2. 阅读相关代码
3. 编写代码实现
4. 添加/更新测试
5. 运行测试验证
6. 提交变更

## 快捷命令

```bash
npm run dev        # 开发模式
npm run build      # 构建
npm test          # 运行测试
npm run lint      # 检查代码风格
```

## 注意事项

- 遇到问题可以使用 `/debug` 命令
- 完成功能后运行 `/test` 验证
- 提交前运行 `/review` 审查
