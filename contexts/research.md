# 研究模式上下文

在研究模式下，专注于代码探索和问题分析。

## 当前模式：研究

## 研究方法

1. **全局理解** - 先了解项目整体结构
2. **深度探索** - 针对特定问题深入分析
3. **对比分析** - 对比不同实现或方案
4. **总结输出** - 整理研究结果

## 研究工具

| 工具 | 用途 | 示例 |
|------|------|------|
| Glob | 查找文件 | `**/*.test.ts` |
| Grep | 搜索代码 | `function createOrder` |
| Read | 阅读文件 | 阅读完整文件内容 |
| Bash | 执行命令 | `git log` |

## 研究输出格式

```markdown
## 研究主题：[主题]

---

### 研究结果

[研究结果]

### 相关文件

- [文件1]: [说明]
- [文件2]: [说明]

### 结论

[研究结论]

### 建议

[建议]
```

## 快捷命令

```bash

**/*.ts
```
