## 核心 Agents

### interviewer

```
文件: agents/interviewer.md
模型: opus
职责: 需求访谈专家
阶段: 访谈
```

**核心能力:**
- 系统化访谈理解用户需求
- 8 维度完整度检查
- KANO 需求分析模型
- 主动发现隐含需求和矛盾

**职责边界:**
- ✅ 执行需求访谈
- ✅ 输出需求摘要
- ❌ 禁止执行架构设计
- ❌ 禁止执行代码编写

**触发方式:**
```bash
/interview
```

**输出:**
- 需求摘要文档
- KANO 需求分类表
- 8 维度完整度状态
