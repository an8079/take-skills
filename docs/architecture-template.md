## 7. 性能设计

### 7.1 性能目标

| 指标 | 目标值 |
|------|--------|
| API 响应时间 | < 200ms (P95) |
| 吞吐量 | > 1000 QPS |
| 并发连接 | > 1000 |
| 数据库查询 | < 50ms |

- 缓存：Redis 缓存热点数据
