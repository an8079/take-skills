## 记忆系统架构

### Memory Bank 结构

```
memory-bank/
├── 项目进展.md        # 项目状态跟踪
│   ├── 项目概览
│   ├── 当前阶段
│   ├── 任务进度
│   ├── 变更日志
│   ├── 待解决问题
│   └── 已解决问题
│
├── 学习记录.md        # 用户反馈学习
│   ├── 待处理的纠正
│   ├── 已应用的规则
│   └── 项目规则
│
└── 技术决策.md        # 技术选择记录
    ├── 技术栈选择
    ├── 架构决策
    ├── 数据模型决策
    └── API 设计决策
```

### 记忆写入流程

```
事件触发
    │
    ├────▶ 代码修改 ───▶ 更新项目进展.md
    │
    ├────▶ 任务完成 ───▶ 更新任务进度表
    │
    ├────▶ 用户纠正 ───▶ 记录到学习记录.md
    │                        │
    │                        ▼
    │                  /reflect 触发 ───▶ 应用规则
    │
    ├────▶ 技术决策 ───▶ 记录到技术决策.md
    │
    └────▶ 问题解决 ───▶ 移动到已解决问题
```

## 上下文注入系统

### 上下文切换流程

```
用户命令
    │
    ▼
命令解析
    │
    ▼
阶段识别
    │
    ▼
加载对应上下文
    │
    ▼
注入到 AI 提示词
```

### 上下文类型

| 上下文 | 触发条件 | 内容 |
|--------|----------|------|
| dev.md | 编码阶段 | 编码规则、最佳实践 |
| review.md | 审查阶段 | 审查标准、检查清单 |
| research.md | 研究阶段 | 研究方法、资料来源 |
| delivery.md | 交付阶段 | 交付标准、检查清单 |
